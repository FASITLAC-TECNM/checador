TÉRMINOS Y CONDICIONES DE USO: SISTEMA DE CONTROL DE ASISTENCIA
Última actualización: 16 de febrero de 2026

Los presentes Términos y Condiciones regulan el acceso y uso del software de control de asistencia (en adelante, "el Sistema"), desarrollado para la gestión de jornada laboral y registro de personal. Al utilizar el Sistema, el Usuario y la Entidad contratante aceptan cumplir con las siguientes cláusulas:

1. OBJETO DEL SISTEMA
El Sistema es una herramienta tecnológica diseñada para automatizar el registro de entradas, salidas y permanencia del personal mediante tecnologías de reconocimiento facial, captura de imagen y gestión de datos en bases de datos relacionales.

2. RECONOCIMIENTO FACIAL Y USO DE IMAGEN
Para garantizar la integridad del registro, el Sistema utiliza el procesamiento de imágenes mediante la biblioteca Face-api.js.

Finalidad: La imagen capturada (selfie con marca de tiempo) tiene el único propósito de validar la identidad del usuario en el momento del registro.

Procesamiento: El sistema genera un vector biométrico o utiliza la imagen para comparar rasgos faciales. El usuario otorga su consentimiento explícito para que su imagen sea procesada con fines de control de asistencia.

Almacenamiento: Las imágenes o datos derivados se almacenarán de forma segura en nuestros servidores de base de datos (PostgreSQL/MySQL), limitando el acceso solo a personal administrativo autorizado.

3. RECOPILACIÓN Y ACCESO A LA INFORMACIÓN
El Sistema recopila y procesa la siguiente información del usuario:

Datos Identificativos: Nombre completo, número de empleado, y "Nombre de Nodo" (parámetro general de la terminal de registro).

Registros de Tiempo: Fecha, hora exacta y ubicación (si aplica) del registro.

Información Técnica: Datos del dispositivo desde el cual se realiza el acceso (PC con Electron o navegador).

4. PRIVACIDAD Y SEGURIDAD DE DATOS
Nos comprometemos a proteger la información conforme a las mejores prácticas de seguridad informática:

La base de datos cuenta con controles de acceso restringidos.

Los datos no serán compartidos, vendidos ni cedidos a terceros ajenos a la relación laboral entre el Usuario y la Entidad, salvo por requerimiento legal.

5. RESPONSABILIDADES DEL USUARIO
El Usuario se compromete a:

Realizar el registro de manera personal e intransferible.

No intentar vulnerar los sistemas de reconocimiento facial mediante el uso de fotografías, máscaras o cualquier método de suplantación de identidad.

Proporcionar información verídica al configurar su perfil.

6. CONFIGURACIÓN DEL SISTEMA
El administrador del sistema es responsable de la correcta configuración de los "Nodos" y terminales. El campo "Node Name" es un parámetro general que identifica la ubicación física o lógica del punto de control y debe ser configurado previo al uso de dispositivos de cámara o huella dactilar.

7. PROPIEDAD INTELECTUAL
El código fuente, diseño de la interfaz (incluyendo el dashboard minimalista y logotipos), y los algoritmos de integración son propiedad exclusiva del desarrollador. Queda prohibida la reproducción total o parcial del software sin autorización previa.

8. LIMITACIÓN DE RESPONSABILIDAD
El desarrollador no se hace responsable por fallos en el registro derivados de:

Mala conexión a internet del usuario.

Condiciones de iluminación inadecuadas que impidan el funcionamiento de Face-api.js.

Uso indebido de las credenciales de acceso.

Aceptación
Al marcar la casilla "Acepto los Términos y Condiciones" o al realizar su primer registro de asistencia, el usuario manifiesta haber leído, comprendido y aceptado la totalidad de este documento.
